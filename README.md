Данная папка представляет собой сборник учебных, а также пет-проектов 
в сфере Data Science.
Каждый из проектов оформлен, поэтому можно просто прочитать
введение, описание и цель.

В проектах используется широкий спектр инструментов по работе с данными:
- Подгрузка и предобработка данных
- Качественный и количественный анализ данных (графики, зависимости, корреляции)
- Подготовка данных к машинному обучению (нормализация, кодирование, пайплайны и прочее)
- Непосредственно машинное обучение и получение предсказаний
- Оценка моделей различными метриками в зависимости от задачи
